# 💬 Quote Generator API

A Flask-based web application that fetches and displays inspirational quotes using a public REST API. This project demonstrates API integration, JSON data handling, and backend web development using Python and Flask.

---

##  Project Overview

The Quote Generator API application connects to an external Quotes API and displays a random inspirational quote each time the user requests one. It is a simple project built to understand how REST APIs work and how Flask communicates with external services.

---

##  Features

* 🎯 Generate random inspirational quotes
* 🌐 External REST API integration
* 🔄 Fetch new quote with a single click
* 📦 JSON data processing
* 🖥️ Simple and responsive user interface
* ⚡ Fast API response

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yuktapatil18/quote_generator_api.git
```

### 2. Navigate to the project

```bash
cd quote-generator-api
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the application

```bash
python app.py
```

### 7. Open your browser

```text
http://127.0.0.1:5000
```
